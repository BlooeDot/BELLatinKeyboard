# BELLatinKeyboard
Windows Keyboard Layout for Belarus Latin writing
## Jak pisać
### š ž č ū
`Left Alt` + `s` = `š`

### ś ź ć ń ł
`'` zatym `s` = `ś`
> UVAGA:
> Kali vy nacisnicie `'`, to na ekranie ničoga nie adbudzicca. Alie, kali wy dadadzicie `s`, `z`, `c`, `n`, `l` to budzi nadrurovanna adpaviednaja litara z zmiagcenniem.
